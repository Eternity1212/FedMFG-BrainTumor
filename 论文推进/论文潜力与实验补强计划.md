# FedMFG 论文潜力与实验补强计划

更新时间：2026-06-22

## 一、项目当前定位

本项目研究方向是：**模态缺失与模型异构条件下的多中心脑肿瘤 MRI 联邦分类诊断**。

主方法 `FedMFG` 不是单纯做“缺失模态分类”，也不是普通联邦学习，而是同时处理：

- 不同客户端 MRI 模态不一致。
- 2D 图像客户端和 3D 体数据客户端并存。
- 客户端模型主体不适合直接参数平均。
- 类别分布和标签覆盖不完全一致。

当前最适合讲成一篇“医学影像联邦学习 + 缺失模态 + 模型异构”的方法论文。

## 三、最接近论文与差异

### 1. FedAMM

论文：[FedAMM: Federated Learning for Brain Tumor Segmentation with Arbitrary Missing Modalities](https://papers.miccai.org/miccai-2025/paper/1764_paper.pdf)

它是当前最接近本项目的论文。共同点是都处理医学影像联邦学习中的任意模态缺失，也都使用模态组合相关的原型信息。

关键差异：

- FedAMM 面向脑肿瘤分割，本项目面向脑肿瘤分类。
- FedAMM 主要处理模态缺失，本项目额外强调 2D/3D 模型结构异构。
- 本项目提出原型一致性指导的分类头聚合，而不是直接完整模型聚合。

风险：

- 仅说“模态组合原型”不够新，因为 FedAMM 已经很接近。
- 论文必须把“模型异构 + 分类头可靠聚合 + 多源分类任务”讲清楚，避免被认为只是 FedAMM 的分类版。

### 2. FedMEMA / FedMEPD

相关链接：[FedMEMA AAAI 页面](https://ojs.aaai.org/index.php/AAAI/article/view/27909)，[FedMEPD arXiv](https://arxiv.org/abs/2603.04887v1)

这类方法使用模态专属编码器和多模态锚点，重点是个性化分割和缺失模态表示校准。

本项目可借鉴：

- 论文结构上强调“inter-modal heterogeneity”。
- 实验要展示不同模态组合客户端的表现。
- 可视化可以做特征空间聚类或原型相似度热图。

### 3. PmcmFL

论文：[Multimodal Federated Learning with Missing Modality via Prototype Mask and Contrast](https://arxiv.org/abs/2312.13508)

它强调使用原型作为缺失模态的掩码，并加入对比学习。

本项目可借鉴：

- 原型库如何作为全局先验。
- 缺失模态比例变化下的鲁棒性实验。
- 单模态推理或少模态推理实验。

### 4. MFCPL

论文：[Cross-Modal Prototype based Multimodal Federated Learning under Severely Missing Modality](https://arxiv.org/html/2401.13898)

它强调跨模态原型和严重模态缺失。

本项目可借鉴：

- 增加不同缺失程度设置，例如 25%、50%、75% 模态缺失。
- 把“严重缺失”作为实验场景之一。

## 四、当前实验结果判断

毕业论文草稿记录的主实验：

| 方法 | Accuracy | Macro F1 |
| --- | ---: | ---: |
| Solo | 84.25 | 84.05 |
| FD | 80.79 | 76.62 |
| FedGH | 73.18 | 68.28 |
| FedProto | 83.95 | 75.93 |
| FedTGP | 87.63 | 87.01 |
| FedMM | 78.58 | 77.50 |
| FedAMM | 85.91 | 85.27 |
| FedMFG | 90.25 | 90.29 |

从数字上看，FedMFG 比最强基线 FedTGP 高约 2.62 个百分点 Accuracy，比 FedAMM 高约 4.34 个百分点 Accuracy，具备写论文的潜力。

但当前最大问题是：代码仓库中的 `fedmfg_history.json` 没有保存最终测试结果，只保存了验证集曲线。投稿前不能只依赖 Word 文档里的表格，必须重新跑并保存正式结果。

## 九、论文写作建议

推荐论文标题方向：

```text
FedMFG: Modality-aware Fusion and Prototype-guided Head Aggregation for Heterogeneous Federated Brain Tumor Classification with Missing MRI Modalities
```

推荐贡献写法：

1. 提出一个面向多中心脑肿瘤分类的异构联邦学习框架，统一处理 MRI 模态缺失、2D/3D 模型结构异构和类别分布异构。
2. 设计模态感知特征融合模块，使客户端只利用真实可用模态，避免零填充或生成补全带来的分布偏差。
3. 构建模态组合级类别原型库，并通过教师原型约束实现跨客户端语义对齐。
4. 提出原型一致性指导的分类头聚合策略，在不聚合异构特征提取器的情况下提升全局分类边界稳定性。
5. 在多源脑肿瘤 MRI 分类任务上验证 FedMFG 相比 FedAMM、FedTGP、FedProto 等方法的优势。
